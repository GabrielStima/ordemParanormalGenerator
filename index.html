Gerador para Ordem Paranormal

Esse projeto tem como motivação ajudar/facilitar os jogadores de Ordem Paranormal.

Todas as regras aplicadas nos códigos são baseadas no livro basico, regras como:

  • Regra de atributos e pericias por nivel de NEX
  • Indicação de como criar uma missão
  • etc...

Como todo o projeto usa informações aleatórias pode ser que tenha algumas informações conflitantes, como por exemplo traços de personalidade completamente opostas, porem peço que relevem, a ideia é usar os geradores para ter como base e poder imaginar em cima da ideia proposta.

Adicionei alguns itens novos nas tabelas de geração e algumas tabelas adicionais, tanto para formar uma missão quanto para formar um agente.
